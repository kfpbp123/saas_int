# 📦 MineBot TG — Система управления контентом для Minecraft каналов

Комплексное решение для автоматизации Telegram-каналов о модах Minecraft. Бот сочетает в себе ИИ-генерацию постов, систему очередей, водяные знаки и юзербота-сканера.

## 🚀 Основные возможности
- **🧠 AI Post Generation:** Автоматическое создание описаний модов через Gemini 2.0 Flash или Groq (Llama 3.3).
- **⏳ Smart Queue:** Умная очередь публикаций с настраиваемым интервалом (по умолчанию каждые 8 часов).
- **🛰️ Userbot Bridge:** Юзербот сканирует сторонние каналы, находит новые файлы и пересылает их основному боту.
- **🎨 Auto-Watermark:** Автоматическое наложение водяного знака на изображения модов.
- **📊 Comments Analysis:** Сбор комментариев из привязанной группы и их анализ с помощью ИИ для поиска идей.
- **⚙️ Admin Control:** Полное управление через бота: включение/выключение автопостинга, ручное создание постов, управление очередью.
- **📱 Web App Integration:** Поддержка Telegram Mini App для визуального управления.

## 🛠 Установка и запуск

1. **Клонируйте репозиторий**
2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Настройте `.env`:**
   Укажите `TELEGRAM_TOKEN`, `API_ID`, `API_HASH`, `GEMINI_KEY` и другие ключи.
4. **Запустите бота:**
   ```bash
   python main.py
   ```
5. **Запустите юзербота:**
   ```bash
   python userbot_worker.py
   ```

### [v1.3.0] — 2026-03-15
**Добавлено:**
- **Progress Bar Animation:** Текстовая анимация загрузки (⏳) для всех длительных процессов (анализ, тренды, генерация).
- **Улучшенная аналитика:** Увеличен объем выборки комментариев до 100 сообщений для более глубокого понимания запросов аудитории.
- **Многопоточность:** Анимация работает в фоновом режиме, не блокируя логику бота.

### [v1.2.0] — 2026-03-15
**Добавлено:**
- **Web-Searching (Trends):** Новая кнопка в меню для поиска популярных модов на CurseForge и трендовых видео на YouTube.
- **Интеграция с CurseForge:** Автоматический скрапинг популярных дополнений.
- **YouTube Trends:** Поиск актуальных обзоров модов.

### [v1.1.0] — 2026-03-15
**Добавлено:**
- **Глобальный контроль автопостинга:** Кнопка в меню настроек для мгновенного включения/выключения автоматики.
- **Синхронизация юзербота:** Юзербот теперь проверяет статус автопостинга в базе данных перед началом сканирования.
- **Уникальные ID файлов:** Переход на `file_unique_id` для 100% защиты от дубликатов в очереди и канале.
- **Логирование комментариев:** Вывод в консоль при сохранении новых сообщений из групп.

**Исправлено:**
- **Баг ручного создания:** Исправлена проблема, при которой бот молчал после нажатия кнопки "Создать".
- **Пустые посты:** Добавлены проверки на ошибки ИИ; теперь посты не создаются без текста.
- **Контекст ИИ:** Увеличено количество учитываемых комментариев (до 30) для более точного анализа сообщества.
