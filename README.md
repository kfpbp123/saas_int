# 📦 MineBot TG — Система управления контентом для Minecraft каналов

Комплексное решение для автоматизации Telegram-каналов о модах Minecraft. Бот сочетает в себе ИИ-генерацию постов, систему очередей, водяные знаки и юзербота-сканера.

## 🚀 Основные возможности
- **🧠 AI Post Generation:** Автоматическое создание описаний модов через Gemini 2.0 Flash или Groq (Llama 3.3).
- **⏳ Smart Queue:** Умная очередь публикаций с настраиваемым интервалом (по умолчанию каждые 8 часов).
- **🛰️ Userbot Bridge:** Юзербот сканирует сторонние каналы, находит новые файлы и пересылает их основному боту.
- **🎨 Auto-Watermark:** Автоматическое наложение водяного знака на изображения модов.
- **📊 Comments Analysis:** Сбор комментариев из привязанной группы и их анализ с помощью ИИ для поиска идей.
- **⚙️ Admin Control:** Полное управление через бота: включение/выключение автопостинга, ручное создание постов, управление очередью.
- **📱 Web App Integration:** Поддержка Telegram Mini App для визуального управления.

## 🛠 Установка и запуск

1. **Клонируйте репозиторий**
2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Настройте `.env`:**
   Укажите `TELEGRAM_TOKEN`, `API_ID`, `API_HASH`, `GEMINI_KEY` и другие ключи.
4. **Запустите бота:**
   ```bash
   python main.py
   ```
5. **Запустите юзербота:**
   ```bash
   python userbot_worker.py
   ```
